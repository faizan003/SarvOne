VC Revoke Test Interface

This interface allows you to test the VC revoke functionality.

Instructions:

  1. Enter a valid VC ID (you can find this in the organization dashboard)
  2. Optionally enter a revocation reason
  3. Click "Revoke VC" to test the functionality
  4. The result will show below

Expected Behavior:

  • If blockchain is configured: VC will be revoked on blockchain and locally
  • If blockchain is not configured: VC will be revoked locally only with a warning
  • If VC doesn't exist or you don't have permission: Error message will be shown
