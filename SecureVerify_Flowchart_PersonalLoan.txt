================================================================================
                              DATA FLOW SUMMARY
================================================================================

USER (John Doe):
• Registers → Gets DID → Receives Salary VC → Applies for Loan → Views Access History

ORGANIZATION (Infosys):
• Registers as Employer → Issues Salary Certificate → Signs VC → Uploads to IPFS → Anchors on Blockchain

VERIFIER (HDFC Bank):
• Registers as Bank → Looks up John's DID → Verifies VC → Checks Blockchain → Approves Loan → Logs Access

SECURITY FEATURES:
• Multi-factor Authentication (Selfie + Aadhaar)
• Digital Signatures (RSA-2048)
• Blockchain Anchoring (Immutable)
• IPFS Storage (Decentralized)
• Access Logging (Transparency)
• Role-based Permissions (Bank sees only salary data)

PRIVACY FEATURES:
• Granular Access Control (Bank can't see education/health data)
• Complete Audit Trail (John sees who accessed what)
• User Consent (John controls data sharing)
• Encrypted Storage (Private keys protected)
